# A+B 问题

## 题目描述

给定两个整数 a 和 b，计算它们的和。

## 输入格式

一行，包含两个整数 a 和 b，用空格隔开。

## 输出格式

一行，包含一个整数，表示 a+b 的结果。

## 示例

**输入：**
```
1 2
```

**输出：**
```
3
```

## 数据范围

- -1000 ≤ a, b ≤ 1000

## 思路

这是最基础的输入输出题，用来熟悉 C++ 的基本结构：
- `#include <iostream>` — 引入输入输出库
- `using namespace std;` — 使用标准命名空间
- `cin >> a >> b;` — 读取输入
- `cout << a+b << endl;` — 输出结果

## 代码

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cin >> a >> b;
    cout << a + b << endl;
    return 0;
}
```

## 复杂度分析

- 时间复杂度：O(1)
- 空间复杂度：O(1)

## 总结

这道题帮你回忆 C++ 的基本结构：
1. 头文件引入
2. 命名空间
3. main 函数
4. 输入输出语句
